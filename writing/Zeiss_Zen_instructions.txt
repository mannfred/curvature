Zen (Zeiss) software instructions from MacGregor (Spring 2019)

Camera tab - microscope components - Stemi508

Objective - 1x
Zoom - whatever zoom you're on
camera adapter - 0.5x

lever in back of stereo microscope controls camera ON/OFF

"LIVE" button - active camera - Axiocam105

Camera tab - camera menu - set exposure
White balance - auto

SNAP- takes photo

Graphics - scale bar - right click - format graphic elements

Tools - scaling (to change scale bar)

If you change zoom you have to "assign new scaling to image" by going to TOOLS - scaling

Processing tab - method - export/import - image export - parameters - apply

Need to EXPORT not "Save" to keep overlayed graphics (eg scale bars). Easiest to go to File - export
